## deduplicate-images
A small project in Python to remove duplicate images.

# Reason for this project
I wanted to exercise my Python skills and I needed a way to remove duplicates of images. Of course as a software developer the solution is either to find a software to do this for me or write my own!

Below are the steps I took to make this small project.

<b> Step 1: </b>
Find a way to identify duplicates reguardless of cropping.

<b> Step 2: </b>
Make sure that we don't have A, B and B, A (the same pair but in a different order).

<b> Step 3: </b>
Let the user as the human go through the duplications.

<b> Step 4: </b>
Create user friendly functionality (saving, going back)

<b> Step 5: </b>
Optimization of hashing images

<b> Step 6: </b>
Create a exe file to make this user friendly and not require a CLI. This needs to be something that someone without a technical background can download and use.
